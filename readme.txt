A WordPress plugin to auth users from TotalKYC APIs. 
 
== Description ==
 
TotalKYC plugin APIs allow you to authenticate 15+ acceptable KYC documents in real-time directly through Government Sources.
 
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
